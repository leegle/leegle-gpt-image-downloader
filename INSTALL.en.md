# leegle-gpt-image-downloader English Illustrated Installation Guide

This guide is designed for beginners who are installing a Chrome extension for the first time.  
Follow the steps below to install and use `leegle-gpt-image-downloader` to batch download ChatGPT generated images.

---

## 01 Download the Extension Package

Open the GitHub project page and click **Releases**.

![Download the extension package](docs/images/01-download-release.png)

Find the latest version, for example:

```text
v0.0.18
```

Download the file:

```text
leegle-gpt-image-downloader-v0.0.18.zip
```

---

## 02 Extract the ZIP File

After downloading, right-click the ZIP file and choose:

```text
Extract All
```

![Extract ZIP file](docs/images/02-extract-zip.png)

After extraction, you will get a folder such as:

```text
leegle-gpt-image-downloader-v0.0.18
```

Note: Chrome cannot load the ZIP file directly. You must extract it first.

---

## 03 Open the Chrome Extensions Page

Open Chrome and enter the following address in the address bar:

```text
chrome://extensions/
```

Then press Enter.

![Open Chrome extensions page](docs/images/03-open-extensions.png)

---

## 04 Enable Developer Mode

On the Extensions page, find **Developer mode** in the upper-right corner and turn it on.

![Enable Developer mode](docs/images/04-enable-developer-mode.png)

After enabling Developer mode, you will see the **Load unpacked** button.

---

## 05 Load the Unpacked Extension

Click:

```text
Load unpacked
```

![Load unpacked extension](docs/images/05-load-unpacked.png)

Select the extracted extension folder:

```text
leegle-gpt-image-downloader-v0.0.18
```

![Select extension folder](docs/images/06-select-folder.png)

Important: Select the folder, not the ZIP file.

---

## 06 Confirm the Extension Was Installed

If the installation is successful, you will see this extension on the Chrome Extensions page:

```text
leegle-gpt-image-downloader
```

![Extension installed](docs/images/07-extension-installed.png)

---

## 07 Pin the Extension Icon

For easier access, pin the extension to the Chrome toolbar.

Steps:

1. Click the puzzle icon in the upper-right corner of Chrome.
2. Find `leegle-gpt-image-downloader`.
3. Click the pin icon next to it.

![Pin extension icon](docs/images/08-pin-extension.png)

---

## 08 Open a ChatGPT Image Conversation

Open ChatGPT and enter a conversation that contains generated images.

Find the image group you want to download and scroll it to the center of the screen.

![Open ChatGPT image conversation](docs/images/09-open-chatgpt-image.png)

Recommended checks:

- The large image on the left is visible.
- The thumbnail list on the right is visible.
- The images are fully loaded.
- The page is not showing a blank loading state.

---

## 09 Switch Between Chinese and English

There is an `EN / 中文` button in the upper-right corner of the extension window.

- If the interface is in Chinese, click `EN` to switch to English.
- If the interface is in English, click `中文` to switch back to Chinese.
- The language setting is saved locally in your browser, so the extension will remember your last selected language.

---

## 10 Auto-Capture Images

Click the extension icon in the upper-right corner of Chrome.

The extension window will open and automatically capture the currently visible high-resolution image group.

![Auto capture images](docs/images/10-auto-capture.png)

After capturing, the extension will show how many images were collected, for example:

```text
Captured 6 images
```

---

## 11 Download the High-Resolution ZIP File

Click:

```text
Download Captured HD ZIP
```

![Download HD ZIP file](docs/images/11-download-zip.png)

The extension will package the images into a ZIP file.

Example filename:

```text
leegle-gpt-image_20260619_2315.zip
```

---

## 12 View the Extracted Images

After downloading, right-click the ZIP file and extract it.

Example structure for one image group:

```text
leegle-gpt-image_20260619_2315/
  image_01.png
  image_02.png
  image_03.png
  image_04.png
  image_05.png
  image_06.png
```

![View downloaded images](docs/images/12-view-images.png)

The extension uses English filenames to avoid garbled text after extraction on Windows.

# FAQ

## 1. Why does Chrome say the extension cannot be loaded?

Make sure you selected the extracted extension folder, not the ZIP file.

Correct:

```text
leegle-gpt-image-downloader-v0.0.18
```

Incorrect:

```text
leegle-gpt-image-downloader-v0.0.18.zip
```

## 2. Why were no images detected?

Please check:

1. You are on a ChatGPT conversation page.
2. The page contains images generated by ChatGPT.
3. The target image group is scrolled to the center of the screen.
4. The images are fully loaded.

If it still does not work, click:

```text
Recapture Visible HD Group
```

## 3. Why did it download only one image?

If the page only shows one large image and does not show the thumbnail list on the right, the extension may only detect the current large image.

Scroll the full image group into the center of the screen and make sure both the large image and the right-side thumbnails are visible, then capture again.

## 4. Why is the filename not based on the conversation title?

To avoid garbled text when extracting ZIP files on Windows, the extension uses an English filename format:

```text
leegle-gpt-image_YYYYMMDD_HHMM.zip
```

This is expected behavior.

## 5. How do I update the extension?

When a new version is released, for example:

```text
v0.0.19
```

Update steps:

1. Download the latest ZIP file.
2. Extract the ZIP file.
3. Open `chrome://extensions/`.
4. Remove or disable the old version.
5. Click **Load unpacked**.
6. Select the latest extension folder.

# Privacy

This extension runs locally in your browser.

It does not upload images, store chat content, or connect to any third-party server.  
Image collection, high-resolution preview, and ZIP packaging are processed locally in your browser.
