## 这个插件解决了什么痛点？

在使用 ChatGPT 生成图片时，经常会遇到一个很麻烦的问题：一次生成多张图片后，需要用户一张一张点击、打开、查看高清图，再手动右键保存。

如果一次只生成 1 张图，问题还不明显。
但如果一个提示词生成 4 张、6 张、10 张，甚至一个会话里连续生成很多组图片，手动下载就会非常浪费时间。

常见痛点包括：

1. **图片需要一张一张下载**
   ChatGPT 生成多张图片后，默认没有批量下载按钮，用户只能逐张打开和保存。

2. **高清图获取麻烦**
   页面上看到的可能是缩略图或预览图，想保存高清原图，需要点击进入大图后再下载。

3. **多组图片容易混乱**
   一个会话里生成多组图片后，手动保存时容易漏掉、重复下载，或者分不清哪一张属于哪个提示词。

4. **文件命名不统一**
   手动下载的文件名可能很乱，不方便后期整理、归档和上传。

5. **时间成本高**
   对经常做 AI 图片、产品图、电商图、设计图、提示词测试的人来说，每天重复下载图片会占用大量时间。

## 这个插件有什么好处？

`leegle-gpt-image-downloader` 是一个专门用于批量下载 ChatGPT 生成图片的 Chrome 插件。

它可以自动识别当前 ChatGPT 会话中的生成图片组，采集高清原图，并一键打包为 ZIP 文件下载。

主要好处：

1. **一键批量下载**
   不需要一张一张保存，插件可以把当前图片组一次性打包下载。

2. **支持高清原图**
   插件优先读取 ChatGPT 图片中的原始 file 图片链接，尽量避免下载低清缩略图。

3. **自动识别当前图片组**
   打开插件后，可以自动采集当前屏幕可见的图片组，适合快速下载刚刚生成的图片。

4. **ZIP 打包更方便**
   多张图片会自动整理到一个 ZIP 文件中，方便保存、传输、上传和归档。

5. **文件名更规范**
   ZIP 文件和内部图片文件使用英文安全格式，例如：

   ```text
   leegle-gpt-image_20260619_2315.zip
   image_01.png
   image_02.png
   image_03.png
   ```

   这样可以避免 Windows 解压后出现中文乱码。

6. **本地运行，更安全**
   插件在本地浏览器中运行，不需要服务器，不上传图片，不保存聊天内容，也不依赖第三方后台。

7. **支持中英文界面**
   插件支持中文和英文界面切换，方便不同用户使用。

## v0.0.18 新增

- 支持高清预览：扫描图片组后，点击任意缩略图即可读取对应 `backend-api/estuary/content?id=file_...` 原始图片并在插件内打开高清大图。
- 支持当前组一键高清查看：点击「查看当前组高清图」会采集当前屏幕中可见图片组的高清原图，并自动打开第一张高清图预览。
- 支持高清 ZIP 下载：采集时优先下载每个缩略图按钮内的原始 file 图片，不使用截图裁剪作为主流程。
- 保留当前会话图片组扫描、采集当前可见高清组、采集选中组高清原图、下载已采集高清 ZIP。

## 推荐使用流程

### 下载当前屏幕这一组 6 张图

1. 打开 ChatGPT 生成图会话。
2. 把目标图片组滚动到屏幕中间，确保左侧大图和右侧小图都能看到。
3. 点击插件图标。
4. 点击「采集当前可见高清组」。
5. 采集完成后点击「下载已采集高清 ZIP」。

### 先看高清图再下载

1. 点击「扫描当前会话图片组」。
2. 在插件列表里点击任意缩略图。
3. 插件会读取对应高清原图并弹出高清预览。
4. 需要下载该图时点「加入下载」，或直接采集整组。

### 下载整个会话多组图

1. 点击「扫描当前会话图片组」。
2. 勾选需要的图片组，或点击「选全部组」。
3. 点击「采集选中组高清原图」。
4. 点击「下载已采集高清 ZIP」。

## 安装

1. 解压 ZIP。
2. 打开 `chrome://extensions/`。
3. 打开「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择 `leegle-gpt-image-downloader-v0.0.18` 文件夹。

## 注意

- 原图链接带登录态和临时签名，如果高清预览或下载失败，请刷新 ChatGPT 页面后重试。
- 本插件不上传图片，不需要服务器，ZIP 在本地浏览器内生成。


### v0.0.18 简化流程
- 点击插件图标后，窗口会自动放大并贴合浏览器高度。
- 插件会自动采集当前屏幕可见的 GPT 图片组。
- 文件名默认使用会话名 + 时间。
- 点击“直接下载已采集高清 ZIP”会直接开始下载。


### v0.0.18 插件名称
插件名称统一为：`leegle-gpt-image-downloader`。


### v0.0.18 文件名规则
- ZIP 文件名固定为：`leegle-gpt-image_YYYYMMDD_HHMM.zip`。
- ZIP 内部只使用英文和数字目录，避免中文提示词或会话名在 Windows 解压后乱码。
- 单组图片直接保存为 `image_01.png`、`image_02.png`。
- 多组图片保存到 `group_01`、`group_02` 等目录。
